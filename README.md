# Свадебное приглашение — Telegram Mini App

Минимальный прототип: одна страница с именем из Telegram и кнопкой «Я приду». Ответы сохраняются на бэкенде (по умолчанию в JSON-файл).

## Стек

- **Клиент:** Vue 3 + Vite, Telegram Web App API
- **Сервер:** Node.js + Express
- **Бот:** Telegraf (запускается вместе с сервером)
- **Хранение:** файл `server/rsvp.json`

## Telegram-бот

Бот запускается вместе с Express. При команде **/start** он отправляет приветствие и кнопку **«Открыть приглашение»**, по нажатию открывается мини-приложение. Команда **/invite** тоже показывает эту кнопку.

**Скрытая команда** (не показывается в меню): **/guests** — выводит список людей, которые идут (имя, с партнёром или нет, дата ответа).

### Настройка бота

1. Создайте бота в [@BotFather](https://t.me/BotFather): отправьте `/newbot`, придумайте имя и username.
2. Скопируйте выданный токен.
3. В папке `server` создайте файл `.env`:

```env
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHI...
APP_URL=https://ваш-домен-или-ngrok-ссылку/   # URL, где открыто мини-приложение (HTTPS)
```

4. Запустите сервер — бот подхватит токен и начнёт отвечать. Без `TELEGRAM_BOT_TOKEN` сервер работает как раньше, бот просто не стартует.

**Важно:** `APP_URL` должен указывать на страницу вашего Vue-приложения по HTTPS (для Telegram Mini App обязателен HTTPS). Для локальной проверки используйте [ngrok](https://ngrok.com): `ngrok http 5173` — и этот HTTPS-URL подставьте в `APP_URL` и в клиентский `VITE_API_URL` при необходимости.

## Запуск локально

### 1. Сервер (и бот)

```bash
cd server
npm install
# при необходимости создайте .env с TELEGRAM_BOT_TOKEN и APP_URL
npm run dev
```

Сервер: http://localhost:3000

### 2. Клиент

```bash
cd client
npm install
npm run dev
```

Клиент: http://localhost:5173

Для локальной разработки клиент по умолчанию шлёт запросы на `http://localhost:3000`. Чтобы указать другой URL API, создайте в `client/` файл `.env`:

```
VITE_API_URL=https://your-api.com
```

### 3. Проверка в Telegram

1. Настройте бота (см. выше) и укажите в `APP_URL` HTTPS-адрес клиента (например, от ngrok).
2. Откройте бота в Telegram и нажмите **Start** или отправьте `/start` — появится кнопка «Открыть приглашение».
3. По нажатию откроется мини-приложение; имя подставится из профиля Telegram.

## API

- **POST /api/rsvp** — сохранить ответ «Я приду»
  - Тело: `{ "name": "Имя Фамилия" }`
  - Ответ: `{ "ok": true }`

- **GET /api/rsvp** — список ответов (для админки/проверки)
  - Ответ: `{ "guests": [ { "name": "...", "date": "..." } ] }`
